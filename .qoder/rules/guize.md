# 自动更新 README 规则

## 规则说明
每当用户提到 README 相关内容或更新项目信息时，自动根据最新内容同步更新 README.md 文档。

## 触发条件
- 用户输入包含 "README" 关键词时
- 用户更新了项目结构、功能说明、安装步骤等信息时
- 用户新增或修改了模块/功能时

## 执行规则
1. **自动检测变更**：识别用户描述中与项目相关的变更内容
2. **同步更新 README.md**：根据变更内容，自动更新 README.md 对应章节，包括但不限于：
   - 项目简介
   - 功能特性
   - 安装与使用说明
   - 目录结构
   - 贡献指南
   - 版本更新日志
3. **保持格式规范**：确保 README.md 格式整洁，符合 Markdown 语法规范
4. **内容一致性**：确保 README.md 内容与实际代码库保持一致

## 注意事项
- 不删除用户已有的重要说明内容，仅在原有基础上进行补充和更新
- 若存在冲突内容，优先保留用户最新描述
- 更新后提示用户已完成 README.md 的同步更新
